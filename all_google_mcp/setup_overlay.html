Actions

Sign in with Google

  1. Click Sign in with Google.
  2. Choose the account Cursor should use.
  3. Click Allow for all permissions.

Session saved as token.json on this Mac only.

Sign out deletes local token.json. Sign in again to reconnect.

Connect Cursor

  1. Click Add to Cursor (updates ~/.cursor/mcp.json).
  2. Quit Cursor completely (⌘Q), then reopen.
  3. Confirm all-google-mcp is enabled in Settings → MCP.

Advanced — Google Cloud project

Google Cloud is already set up for the team build. Use this only for your own OAuth app.
